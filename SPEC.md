## 5. 项目规范

### 自动存档规则
**每次完成任何修改后，必须自动执行Git存档：**
1. 每完成一个功能块或页面组件 → 自动 `git add` + `git commit`
2. 提交信息格式：`[功能名] 描述`（如 `[Hero] 更新首屏动效`）
3. 禁止未存档直接结束会话

### Git操作流程
```bash
git add -A
git commit -m "[模块名] 简短描述"
```
